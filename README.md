# ASI_DATA Description

ASI_DATA represents time-series energy data capturing both solar power generation and electrical load demand.

## Fields

- **Solar (kW):**  
  The amount of power generated by the solar system at a given time, measured in kilowatts (kW).

- **Load (kW):**  
  The total electrical demand or consumption at the same time interval, also measured in kilowatts (kW).

## Usage

This dataset is typically used for:
- Energy analysis  
- Load forecasting   
- Demand reduction  
- Energy dispatch
